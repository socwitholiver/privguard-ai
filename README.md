# PrivGuard AI
PrivGuard AI is an offline, AI-powered desktop application designed to detect, classify, and protect sensitive data in Kenyan documents. It uses NLP, Machine Learning, and encryption to safeguard information in alignment with Kenya’s Data Protection Act (2019).

## Features
- Detects Kenyan IDs, KRA PINs, phone numbers, and financial data  
- NLP-based sensitive data classification  
- Offline processing for privacy  
- Redaction & file encryption tools  
- User-friendly GUI (Tkinter / PyQt5)
