# demo-docs

This folder contains 50 sample files for watch-folder demonstrations.

How to use:
1. Open PrivGuard AI.
2. Go to Watch Folder and choose any folder you want PrivGuard to monitor.
3. Point PrivGuard at an empty intake directory.
4. Copy the files from this folder into that watched directory.
5. Watch the dashboard update as PrivGuard scans, redacts, encrypts, and stores artifacts in the background.

File mix:
- 10 payroll text files
- 10 admission text files
- 10 contract markdown files
- 10 bank review CSV files
- 10 county intake log files

All files are local demo data only and intentionally contain patterns such as Kenyan national IDs, KRA PINs, phone numbers, emails, passwords, and financial references.
